1. Overview

Author:Mayuri Pandey

Data can be found abundantly over the Internet, and there are databases available, which help in the storage and retrieval of those data and information. Nowadays, there is a rapid growth of location-specific devices that need geographical features and spatial data, and there exist severe challenges to maintain such data. Hence in 1997 the Open Geospatial Consortium developed few specifications and set few standards for adding various spatial functionality to the database systems.[1]

Therefore, in this regard, Geospatial Database or Spatial Data has been well-defined as collective sets of stored information which is used to depict any data which is related to or consists of information about a particular location on the Earth’s Surface. The main goal of this report is to understand the concept of geospatial databases in brief and to have a thorough understanding of various tools available, how these tools will help us to optimize data.[2]

2. Geospatial Databases

Author:Mayuri Pandey

Geospatial database, defined as a group of different data sets stored in a common System file folder. The database can be generalized as containing geographic information of countries, cities, different administrative divisions,etc. It mainly has data about the locations, geographic features and the relationship between them, which are stored as coordinates and topology. It is structured in a way as to ensure maximum productivity from the database.[2]

It follows ESRI standard format and acts as collective storage for various data types like Raster, Vector, Tables, etc. Another important feature is also the scalability, i.e, single to multiple editing can be achieved. It acts as a physical store for the geographical information using a database stem where we can not only access but also work with the physical instances such as a collection of the data sets using SQL, it consists of a model that represents and manages the geographical information which supports the rules to manage spatial integrity and the tools to work with the attributes.[2]

4. Tools

Author:Mayuri Pandey

There exist various tools or software for analyzing and computing different patterns, which assists the user to understand them more precisely. The functionality of each of them is listed below: [4]

Falconview: This application supports multiple platform mapping functions. It has the ability to showcase a wide range of maps including varied types of geographically referenced overlays.It lacks spatial analysis and also it has limited cartography and mapping tools. Data editing and management are quite difficult in this software.[4][5]

GeoDa:It is free software that has the capability to conduct an analysis of spatial data, spatial modeling, etc. It mainly deals with geostatistics and as an introductory tool for beginners to work with spatial data. This tool is promising in various sectors such as health care, economic development, etc.[4][6]

GrassGIS:This software is designed for the management of land and planning of the environment. It is a free software that is valued for its various uses in sectors concerning analysis, manipulation of terrain, image processing, etc. GrassGIS supports 3D raster analysis.[4][7]

ILWIS:It is an open source software for wide and easy access for users.Some of the basic areas of operation for this software are editing, digitization and displaying geographic data. Further the software also facilitates remote sensing with tools for image enhancement, classification and band manipulation.[4][8]

MapWindows:The software is not a complete software for utilisation in the Geo spatial data processing of the intended tasks such as identity features, map viewer, processing tools and print layout. This software is free to use.[4][9]

QGIS It is basically a geo information system, which enables users to not only examine and change the spatial data but also create and exchange pictorial maps. A plus point of QGIS is that it can handle shapefiles, dxf, coverages, PostGIS formats. Georeferencing of images is also possible with this software.[4][10]

gvSIG:This application is desktop-oriented and has the ability to store, analyze, capture any geographical referenced information to resolve complicated issues. It is also counted as the base 3D visualization tool available having the most interactive user-friendly UI.[4][11]

SagaGIS:Originally designed for analysis of terrains such as watershed extraction, hill shading, and visibility analysis. The tools facilitate operations with multiple windows to lay forward all the analysis such as maps, scattered plots, histograms, etc besides providing methods related to geosciences to the geoscientific community.[4][12]

uDig:This application is free to use having an intuitive user-friendly UI, It is a desktop-oriented application that is supported by all operating systems and also provides the framework on which analytical problems can be resolved.[4][13]

Each tool has its own flexibility of functionalities, few of them are briefly explained below:

Supported by all Operating Systems: There are various operating systems available such as Windows, macOS , Linux and, Android. Each software or tool has been designed such that it can give the user flexibility to use it on these platforms.[14]

Open Source: This criteria states that whether the tool or software in use is freely available or it is a paid version where the user needs to pay a certain amount to use the tool. [14]

Cartographical operations: The operations let the user zoom and pan the extensions of the map and restructure the map components i.e, Map legend, north arrow, scale bar, and inset maps to name a few. [14]

Geoprocessing: Geoprocessing can be defined as an operation that uses a dataset as an input and performs operations on that specific dataset, hence returning the operation as an output dataset as result. [14]

User Friendly: The goal of having a user-friendly UI is to offer an easy navigation GUI for exporting and importing the data format, manipulate and edit the data and analyze the modified layout of the map.[14]

Data manipulation and Analysis: This is one of the most important functionalities provided by the tools to manipulate the data and have a detailed analysis of the same, hence allowing the user to edit and create their own GIS when they need.[14]

Scripting functionality: It includes performing queries and programming functionalities using database and programming language with which users can script and follow workflow repetitively.[14]

GrassGIS:

GrassGIS is a typical tool used in finding Geographical Information systems. It is specially designed in order to analyze and manage the vector and raster data, model the spatial data, 3D visualization of the map and geoprocessing. [15]It is free and open source software. We chose this tool for a deep understanding of the geospatial databases for many reasons, for instance, there are no limitations of operating systems, it supports Linux, Windows and Mac and also it consists of an intuitive, user-friendly GUI and command line, the backend language used here is Python, while using the software, the user need not need to know about the language and can work by knowing few basic commands in the command line. GrassGIS supports all the SQL interface functionalities.[15][16]

To understand this platform in brief, we need to understand few very important aspects of the tool, these are explained below:

Terrain Analysis: It assists in detailed analysis and interpretation of the various topological features through the Geographical Information System. The different Terrain Analysis includes a slope, aspect, viewshed, shaded surfaces, elevations etc. It also shades the elevation layer we came.[16]

Hydrological Analysis:It assists the user to have a thorough understanding, predicting various aspects and managing the water resources.[16]

Why is Raster Data used?

The raster data can be easily rescaled and masked, hence masking is one of the most important criteria to consider while working with maps. Each location contains useful sensitive data, hence Masking is done when maps are read to protect the user data.[16]

Why is Vector Data used?

Vector Data aids in maintaining the vector maps with clean geometry and maintaining the same, it also helps in analyzing the topological data, so it needs to be considered while working for a specific location.[16]
